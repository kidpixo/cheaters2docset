Please note that the help book is not a complete reference to TextWrangler: for a comprehensive guide to TextWrangler's features and commands, please see the PDF manual, which you can open by choosing "User Manual" in the Help menu.

You can also download a copy of the manual from our web site or view the current release notes online.

If you're new to TextWrangler and aren't sure where to start, you can download a disk image which contains a tutorial in PDF format (along with sample files): Tutorial Package

Finally, the excellent "Take Control of BBEdit" contains a great deal of information that applies to TextWrangler as well. Learn more.
